Question Feature cURL Tests
***

Happy Paths (Module, Lesson AND Quiz must exist before performing CRUD on 'Question'

***
1. Create a new question.   -> EXPECT 201
curl -i -X POST http://localhost:8080/api/modules/1/lessons/1/quizzes/1/questions -H "Content-Type: application/json" -d "{\"text\": \"Create question Test?\"}"

RESULT: HTTP/1.1 201
    Content-Type: application/json
    Transfer-Encoding: chunked
***
2. Get all questions from a Quiz (Retrieving by 1 is not efficient).    -> EXPECT 200
curl -i -X GET http://localhost:8080/api/modules/1/lessons/1/quizzes/1/questions

RESULT: HTTP/1.1 200
    Content-Type: application/json
    Transfer-Encoding: chunked
***
3. Update a question    -> EXPECT 200
curl -i -X PUT http://localhost:8080/api/modules/1/lessons/1/quizzes/1/questions/1 -H "Content-Type: application/json" -d "{\"text\": \"Update Question Test!\"}"

RESULT: HTTP/1.1 200
    Content-Type: application/json
    Transfer-Encoding: chunked
***
4. Delete a question    -> EXPECT 204
curl -i -X DELETE http://localhost:8080/api/modules/1/lessons/1/quizzes/1/questions/1

HTTP/1.1 204
Date: Mon, 29 Sep 2025 00:58:34 GMT
